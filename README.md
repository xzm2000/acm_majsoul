比赛共分为两个阶段：预选赛和决赛

外榜地址：https://xzm2000.github.io/acm_majsoul/

## 预选赛

预选赛共3轮，会按总精算点排位，把人数淘汰至2的整数次幂倍，带分5\%进入决赛。

超出4的整数桌的报名视为替补。

预选赛规则如下：

四人南，起始点25000，无击飞，无西入，每轮随机分组，空缺位置交给电脑
